SIDC builder

Build a hyphenated SIDC string matching the server’s sidc_template layout. Preview uses milsymbol in APP-6 drawing mode with the non-hyphenated SIDC (same as the map client and server app6d data). The library is loaded from this app first (/vendor/milsymbol/milsymbol.js) for offline use; if that fails, jsDelivr is used as a fallback.
